## 🛒 ECommerce Admin Panel

An ASP.NET Core MVC admin panel for eCommerce management.

### Features
- **Dashboard** — Statistics (total products, categories, stock value, low-stock alerts)
- **Product Management** — Add, edit, delete products; search & filter by category/status
- **Category Management** — Add, edit, delete categories (with product-count guard)

### Tech Stack
- ASP.NET Core MVC (.NET 10)
- Entity Framework Core + SQLite
- Bootstrap 5 (responsive admin UI)

### Run Locally
```bash
cd ECommerce
dotnet run
# Navigate to http://localhost:5000/admin
```
